# Commands

## Install

```sh
sudo yum install -y [package]
```

## Update

```sh
sudo yum check-update
```

```sh
sudo yum update -y
```

## Search

```sh
sudo yum search [package]
```

## Show Dependencies

```sh
sudo yum deplist [package]
```

## Repository

```sh
sudo vim /etc/yum.repos.d/[name].repo
```

## Erase

```sh
sudo yum erase [package]
```

## Clean

```sh
sudo yum clean all
```

## Show Installed

```sh
sudo yum list installed
```
