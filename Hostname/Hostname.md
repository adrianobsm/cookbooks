# Hostname

## Commands

### Show Host Name

```sh
hostname
```

```sh
hostname -s
hostname -f
```

```sh
sudo cat /etc/hostname
```

### Set Host Name

```sh
sudo hostnamectl set-hostname [name]
hostnamectl status
```

```sh
shutdown -r
```

###

```sh
sudo vim /etc/hosts
```

```sh
127.0.0.1 [name]
```
