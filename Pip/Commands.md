# Commands

## Show Installed

```sh
pip freeze
```

## Install

```sh
pip install [package]
```

```sh
pip install --no-binary :all: [package]
```

```sh
pip install -r requirements.txt
```

```sh
pip install --user -r requirements.txt
```

## Uninstall

```sh
pip uninstall [package]
```

## Upgrade

```sh
pip install --upgrade pip
```

> Perhaps need try with sudo.
