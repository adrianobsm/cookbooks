# Commands

## Add Box

```sh
vagrant box add [name] [url]
```

## Init

```sh
vagrant init [name]
```

## Up

```sh
vagrant up
```

```sh
vagrant up --provider virtualbox
```

## SSH

```sh
vagrant ssh
```

## Stop

```sh
vagrant halt
```

## Suspend

```sh
vagrant suspend
```

## Destroy

```sh
vagrant destroy
```
