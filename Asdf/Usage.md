# Usage

## Add

```sh
asdf plugin-add [name]
```

## List

```sh
asdf plugin-list
```

## Update

```sh
asdf plugin-update --all
```

## Install

```sh
asdf install [name] [version]
```

## Set Global

```sh
asdf global [name] [version]
```

## Reshim

```sh
asdf reshim [name] [version]
```

## Current

```sh
asdf current
```

## List

```sh
asdf list [name]
```

## Uninstall

```sh
asdf uninstall [name] [version]
```
