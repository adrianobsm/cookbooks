# Java

## Installation

```sh
asdf plugin-add java https://github.com/skotchpine/asdf-java.git
```

## Version

```sh
asdf list-all java
```

```sh
asdf install java [version]
```

## Setting

```sh
asdf global java [version]
```

```sh
asdf list java
asdf reshim java [version]
```

```sh
export JAVA_HOME=$(\asdf where java ${BASH_REMATCH[2]})
```
