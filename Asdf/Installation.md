# Installation

```sh
git clone https://github.com/asdf-vm/asdf.git ~/.asdf --branch v0.6.0
```

## Bash

### OS X

```sh
echo -e '\n# ASDF
. $HOME/.asdf/asdf.sh
. $HOME/.asdf/completions/asdf.bash' >> ~/.bashrc
```

### Linux

```sh
echo -e '\n# ASDF
. $HOME/.asdf/asdf.sh
. $HOME/.asdf/completions/asdf.bash' >> ~/.bash_profile
```

## ZSH

```sh
echo -e '\n# ASDF
. $HOME/.asdf/asdf.sh
. $HOME/.asdf/completions/asdf.bash' >> ~/.zshrc
```
