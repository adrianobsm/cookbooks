# Maven

## Dependencies

### OS X

```sh
brew install coreutils gpg
```

## Installation

```sh
asdf plugin-add maven https://github.com/skotchpine/asdf-maven.git
```

## Version

```sh
asdf list-all maven
```

```sh
asdf install maven [version]
```

## Setting

```sh
asdf global maven [version]
```

```sh
asdf list maven
asdf reshim maven [version]
```
