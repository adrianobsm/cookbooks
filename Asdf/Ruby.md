# Node.js

## Dependencies

### OS X

```sh
brew install coreutils gpg
```

## Installation

```sh
asdf plugin-add ruby https://github.com/asdf-vm/asdf-ruby.git
```

## Version

```sh
asdf list-all ruby
```

```sh
KEEP_SOURCE=1 asdf install ruby [version]
```

## Setting

```sh
asdf global ruby [version]
```

```sh
asdf list ruby
asdf reshim ruby [version]
```
