# Minikube

## Installation

```sh
asdf plugin-add minikube https://github.com/alvarobp/asdf-minikube.git
```

## Version

```sh
asdf list-all minikube
```

```sh
asdf install minikube [version]
```

## Setting

```sh
asdf global minikube [version]
```

```sh
asdf list minikube
asdf reshim minikube [version]
```
