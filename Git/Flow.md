# Flow

## Installation

### OS X

```sh
brew install git-flow-avh
```

### Ubuntu

```sh
sudo apt-get install -y git-flow
```

## Commands

### Init

```sh
git flow init
```

### Feature

#### Start

```sh
git flow feature start [name]
```

#### Start From

```sh
git flow feature start [name] [base]
```

#### Finish

```sh
git flow feature finish [name]
```
